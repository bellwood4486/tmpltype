# ドキュメント

tmpltypeの完全なドキュメントです。

## はじめに

tmpltypeが初めての方はこちらから：

- **[はじめに](getting-started.md)** - インストールから最初のコード生成までのステップバイステップチュートリアル

## リファレンスドキュメント

### ユーザー向け

- **[CLIリファレンス](cli-reference.md)** - 完全なコマンドラインリファレンス
  - すべてのコマンドラインオプション
  - 使用例
  - `go generate`との統合
  - ロギングとデバッグ
  - トラブルシューティングガイド

- **[テンプレート構文](template-syntax.md)** - サポートされるGoテンプレート構文
  - フィールド参照（`.Field`, `.Nested.Field`）
  - 制御フロー（`if`, `with`, `range`）
  - コレクション（スライス、マップ）
  - 型推論ルール

- **[`@param`ディレクティブ](param-directive.md)** - 型ディレクティブの完全リファレンス
  - `@param`を使用するタイミング
  - サポートされる型（プリミティブ、スライス、マップ、構造体、ポインタ）
  - 構文例
  - よくあるパターン

- **[テンプレートグルーピング](template-grouping.md)** - サブディレクトリでテンプレートを整理
  - フラット vs グループ化されたテンプレート
  - 名前空間の生成
  - ファイル構造の例
